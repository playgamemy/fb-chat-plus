== Description ==

Unofficial Addon for Facebook Customer Chat. 

Added useful functions including disable chat in some pages, css class for CTA button to show chat, auto show chat after delay, shake conversation to get attention, etc.

This Plugin is currently under active development to bring the community more functions and settings which is lacked in the official plugin. Request through the ticketing system on the WordPress Plugin support page and we will try to accomodate.

== Frequently Asked Questions ==

= How to add a Call to action (CTA) button that open conversation directly? =

1. Simply add css class "fbcp-open-chat" to the button, conversation will open when clicked.
2. You can also use shortcode [fbcp-cta-button] to add a "Chat with us" button with messenger logo
